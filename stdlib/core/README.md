# B.E.L. Core Library (`core`)

This will contain the `core` crate — the foundation available in `no_std` environments.

Planned modules:
- `core::mem` (size_of, etc.)
- `core::ptr`
- `core::option` and `core::result`
- `core::iter`
- Basic primitives and traits

See `docs/` for the full standard library design.
